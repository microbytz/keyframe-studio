# **App Name**: SketchFlow Studio

## Core Features:

- Interactive Drawing Canvas: Multi-layered digital canvas with customizable brush tools (pen, eraser), color palette, and basic pressure sensitivity using web-native APIs.
- Frame Timeline Management: Intuitive interface for adding, deleting, duplicating, and reordering individual animation frames on a scrollable timeline.
- Onion Skinning: Toggleable feature displaying translucent overlays of preceding and succeeding frames for fluid motion tracing during animation.
- Essential Drawing & Editing Tools: Implement a bucket fill tool for coloring, a lasso tool for selection, and a text tool for adding captions and annotations.
- Animation Playback & Export: Real-time preview of animations with adjustable frames-per-second, and option to export the animation as a GIF file.
- AI Creative Inspirer Tool: A generative AI tool that provides brief character traits or scene ideas based on user-provided keywords, acting as a creative tool.
- Project Save/Load (Local): Ability to save and load animation projects directly within the browser's local storage for easy access.

## Style Guidelines:

- Primary accent color: A subtle, desaturated cool blue (#82C9C9) to highlight interactive elements and convey clarity without distraction.
- Main background color: A very light, muted blue-gray (#EFF2F3) to emulate the look of clean paper or a sketchpad.
- Primary functional color: A dark, subdued slate gray (#454D52) for text, lines, and primary tool icons, akin to pencil or ink on paper.
- The entire user interface will utilize the 'PT Sans' font, a humanist sans-serif chosen for its clean readability and friendly, approachable feel, complementing the 'diary-like' aesthetic.
- Simple, line-based icons with a slightly 'hand-drawn' or unrefined quality to maintain a consistent 'sketchbook' and 'diary' inspired visual language.
- A clean, minimalist layout with the largest possible area dedicated to the drawing canvas, complemented by collapsible or unobtrusive toolbars and a bottom-aligned timeline.
- Subtle, functional animations for UI feedback, such as hover states, active tool selection, and frame transitions, ensuring a responsive but calm user experience.
