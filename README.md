
# SketchFlow Studio

SketchFlow Studio is a hand-drawn 2D animation toolkit built with Next.js and React.

## Features

- **Professional Brush Engine**: 15+ brush types with pressure sensitivity and stabilization.
- **Layer System**: Work across multiple layers per frame with live thumbnails.
- **Custom Brushes**: Create brush tips from images or by drawing directly in the app.
- **Animation Suite**: Frame-by-frame timeline with onion skinning and real-time playback.
- **Project Management**: Save work to browser storage or download/upload project files.
- **Export**: Share your creations by exporting directly to GIF.

## Getting Started

1. Pick a brush and start drawing on the canvas.
2. Use the timeline at the bottom to add or duplicate frames.
3. Toggle "Onion Skinning" in the top bar to see ghosted frames for smoother animation.
4. Click the "Export GIF" button to download your final animation.

Built for animators, by animators.
